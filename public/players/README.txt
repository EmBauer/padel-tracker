Put player images in this folder using these file names:
- p1.png
- p2.png
- p3.png
- p4.png
- p5.png
- p6.png

They are shown next to these players (from defaults/data):
- p1 = Sven
- p2 = Viola
- p3 = Simon
- p4 = Joni
- p5 = Lucca
- p6 = Matty

If a file is missing, initials will be shown as fallback.
